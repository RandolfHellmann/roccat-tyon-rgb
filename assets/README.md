# Mouse photos for the button map

Drop two photos of your Tyon in this folder and the **Tasten** (buttons) page
will use them as an interactive, clickable button map. If the files are missing,
the app falls back to the original drawn schematic, so the repo still works
without them.

## Expected files (JPG or PNG)

| File | View | Buttons it covers |
|---|---|---|
| `tyon_top.jpg`  | straight top-down | left/right click, wheel, Dorsal-Fin rocker, the two top button pairs |
| `tyon_side.jpg` | straight-on left side | the two thumb buttons, the X-Celerator paddle, the thumb pedal |

## Tips for usable photos

- Shoot **straight on** (camera parallel to the mouse) so the buttons aren't
  skewed by perspective.
- Even, soft light; avoid glare on the glossy parts.
- Plain background — a **dark, matte** surface blends best with the app's dark
  theme (white works too).
- Fill the frame; as high resolution as possible.

No editing needed on your side — cropping, scaling, orientation and background
clean-up are done in the build step.
